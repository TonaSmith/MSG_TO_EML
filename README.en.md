# MSG_TO_EML

#### Description
一款完全离线、无需依赖 Outlook 环境的 MSG 转 EML 批量转换工具。基于 PyQt6 构建现代化 UI，专为高保密企业内网环境设计，支持拖拽操作与极速大文件处理。
